рЃЦрЃЋрЃўрЃќрЃў рЃърЃарЃЮрЃЏрЃЮ рЃЎрЃЮрЃЊрЃўрЃЌ РђЊ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌрЃў рЃЎрЃўрЃЌрЃ«рЃЋрЃћрЃЉрЃўрЃЌ

рЃЏрЃўрЃўрЃдрЃћрЃЌ 5% рЃўрЃљрЃюрЃў рЃцрЃљрЃАрЃЊрЃљрЃЎрЃџрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃЏрЃЮ рЃЎрЃЮрЃЊрЃў

рЃгрЃћрЃАрЃћрЃЉрЃў:

  • рЃЦрЃЋрЃўрЃќрЃў рЃерЃћрЃЊрЃњрЃћрЃЉрЃљ 10 рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃАрЃљрЃњрЃљрЃю
  • рЃгрЃљрЃарЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃЏрЃЌрЃљрЃЋрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ рЃърЃарЃЮрЃЏрЃЮ рЃЎрЃЮрЃЊрЃА.
  • рЃърЃарЃЮрЃЏрЃЮ рЃЎрЃЮрЃЊрЃўрЃЌ рЃЏрЃўрЃўрЃдрЃћрЃЉрЃЌ 5%-рЃўрЃљрЃю рЃцрЃљрЃАрЃЊрЃљрЃЎрЃџрЃћрЃЉрЃљрЃА рЃЮрЃюрЃџрЃљрЃўрЃю рЃерЃћрЃЎрЃЋрЃћрЃЌрЃљрЃќрЃћ
  • рЃърЃарЃЮрЃЏрЃЮ рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌрЃ«рЃћрЃџ
  • рЃЌрЃЦрЃЋрЃћрЃю рЃњрЃљрЃЦрЃЋрЃЌ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ 3 рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃћ.
  • рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃърЃљрЃАрЃБрЃ«рЃўрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃћ рЃњрЃљрЃЎрЃџрЃЊрЃћрЃЉрЃљрЃЌ.
  • рЃЌрЃБ рЃЊрЃљрЃЎрЃљрЃарЃњрЃљрЃЋрЃЌ 3 рЃАрЃўрЃфрЃЮрЃфрЃ«рЃџрЃћрЃА РђЊ рЃЦрЃЋрЃўрЃќрЃў рЃАрЃарЃБрЃџрЃЊрЃћрЃЉрЃљ.
  • рЃЦрЃЋрЃўрЃќрЃўрЃА рЃЌрЃљрЃЏрЃљрЃерЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ 3 рЃ»рЃћрЃа рЃДрЃЮрЃЋрЃћрЃџ рЃћрЃарЃЌ рЃЊрЃдрЃћрЃерЃў.
